# Drift — Website

Stop letting things silently slip away.

Landing page and product docs for Drift. The app itself lives in a separate repo.

## What's here

- `index.html` — Landing page with quiz flow (single file, no dependencies)
- `docs/one-pager.md` — Product one-pager explaining the idea

## Hosting

Static site — no build step. Host anywhere:

- **GitHub Pages:** Settings → Pages → select branch
- **Netlify:** Drag and drop at [app.netlify.com/drop](https://app.netlify.com/drop)
- **Vercel:** `npx vercel` in this directory
